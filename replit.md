# Hello User Node.js App

## Overview
A simple Node.js application that displays a greeting message to the user.

## Project Structure
- `index.js` - Main entry point that outputs "Hello User!" to the console

## How to Run
The application runs automatically using the configured workflow, or you can run it manually with:
```
node index.js
```
